输入两个整数序列，第一个序列是栈的入栈序列，且递增有序。请判断第二个序列是否为合理的出栈序列

输入两个整数序列，第一个序列是栈的入栈序列，请判断第二个序列是否为该栈的出栈序列

往Young氏矩阵中插入新的元素，要求保证矩阵的行、列有序性

设计算法，计算二叉树T的最大子树和
